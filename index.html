Highlighted Projects


Quinnipiac Senior Capstone Project:

Perfusion simulation software commissioned by the Quinnipiac University Cardiovascular Perfusion Program as a perfusion training simulator. The application was primarily written in JavaScript using Meteor and MongoDB and will be used in the coming years to teach future students.


Hangman MiniGame:

A simple Java program coded in order to gain a deeper understanding of TCP "Three-way handshake" as it was later made with a client side which allowed it to be used across multiple computers where one would create the word and the other would be able to guess.


SQLite Database Project:

A database application which uses Python to communicate with SQLite to retrieve data about TV series from OMDBd's open-source API. The perfect companion for binge watching a new series or quickly getting caught up on what happened in the previous episode.
